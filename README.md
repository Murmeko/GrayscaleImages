## Installation

GrayscaleImages is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'GrayscaleImages'
:git => 'https://github.com/Murmeko/GrayscaleImages.git'
```
